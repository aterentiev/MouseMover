# MouseMover
Automatic mouse movement preventing PC sleep mode or screensaver

This sketch is based on the Digispark (http://digistump.com/products/1) hardware and uses DigiMouse library, available from the DigiStump repository (http://digistump.com/package_digistump_index.json)
